# Setup Instructions for EME Universal Messenger

## Prerequisites
- Ensure you have the following installed:
  - Node.js (version)
  - npm (version)
  - MongoDB (if applicable)
  - Any other necessary tools

## Backend Setup
1. **Clone the Repository**
   ```bash
   git clone https://github.com/JOHNMARK50/EME-Universal-Messenger.git
   cd EME-Universal-Messenger/backend
   ```
   
2. **Install Dependencies**
   ```bash
   npm install
   ```

3. **Set Environment Variables**
   - Create a `.env` file and configure the necessary variables (if applicable).

4. **Run the Server**
   ```bash
   npm start
   ```

5. **Test the API**
   - Use Postman or cURL to test the API endpoints.

## Frontend Setup
1. **Navigate to the Frontend Directory**
   ```bash
   cd EME-Universal-Messenger/frontend
   ```

2. **Install Dependencies**
   ```bash
   npm install
   ```

3. **Run the Development Server**
   ```bash
   npm start
   ```

4. **Access the Application**
   - Open your browser and go to `http://localhost:3000`.
