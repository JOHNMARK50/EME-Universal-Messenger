# EME Universal Messenger

## Project Overview
The EME Universal Messenger is a versatile messaging application designed to unify communications across various platforms, enabling seamless interactions and enhancing productivity.

## Setup Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/JOHNMARK50/EME-Universal-Messenger.git
   ```
2. Navigate to the project directory:
   ```bash
   cd EME-Universal-Messenger
   ```
3. Install the required dependencies:
   ```bash
   npm install
   ```
4. Start the application:
   ```bash
   npm start
   ```

## Feature Roadmap
- **Version 1.0**: Basic messaging functionality across multiple platforms.
- **Version 1.1**: Adding group chat features.
- **Version 1.2**: Implementing end-to-end encryption for user privacy.
- **Version 2.0**: Introducing video and voice call capabilities.
